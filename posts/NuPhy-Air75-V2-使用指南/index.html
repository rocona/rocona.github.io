NuPhy Air75 V2 使用指南

发表于 2026-05-23 分类于 外设

NuPhy Air75 V2 是一把支持三模连接、Mac/Windows 双系统切换、QMK/VIA 改键的 75% 矮轴机械键盘。它的功能不少，但很多操作都藏在 Fn 组合键里，第一次用很容易把蓝牙配对、休眠、截图键、VIA 配置这些东西弄混。

这篇整理 Air75 V2 的日常使用方法：连接设备、切换系统、灯光、休眠、截图、VIA 改键、宏录制和固件更新。

机身开关

Air75 V2 顶部有两个拨动开关。

左侧是系统选择：

  • Win mode：Windows / Linux 使用。
  • Mac mode：macOS 使用。

右侧是连接模式：

  • Power Off：关机。
  • Wired mode：有线模式。
  • Wireless mode：无线模式，包含蓝牙和 2.4G。

如果只是日常打字，按自己的设备选择即可。如果要用 VIA 改键、刷固件，建议切到 Wired mode 并用 USB-C 数据线连接电脑。

设备连接和切换

Air75 V2 的无线设备切换用的是 Fn + 1/2/3/4，注意不是 Fn + F1/F2/F3/F4。

对应关系如下：

  • Fn + 1：蓝牙设备 1。
  • Fn + 2：蓝牙设备 2。
  • Fn + 3：蓝牙设备 3。
  • Fn + 4：2.4G 接收器。

蓝牙配对

  1. 把连接模式切到 Wireless mode。
  2. 短按 Fn + 1、Fn + 2 或 Fn + 3，选择一个蓝牙通道。
  3. 长按对应组合键 3 秒，进入配对模式。
  4. 在电脑、手机或平板的蓝牙列表里选择 Air75 V2。

以后切换设备时，只需要短按对应的 Fn + 数字键。

2.4G 连接

  1. 把 2.4G 接收器插到电脑上。
  2. 把键盘切到 Wireless mode。
  3. 短按 Fn + 4 切到 2.4G 通道。

如果连接不上，可以把接收器拔插一次，再长按 Fn + 4 重新配对。

指示灯含义

Air75 V2 左右两侧都有 RGB 指示灯。日常最有用的是这些状态：

左侧 RGB：

  • 白色：Caps Lock 开启。
  • 黄色：有线模式。
  • 蓝色：蓝牙模式。
  • 绿色：2.4G 模式。

右侧 RGB：

  • 红色：电量 1% - 10%。
  • 橙色：电量 11% - 95%。
  • 绿色：电量 96% - 100%。
  • 绿色闪 3 次：开启休眠模式。
  • 红色闪 3 次：关闭休眠模式。
  • 蓝色标记：Win mode。
  • 白色标记：Mac mode。

看到右侧红灯时就该充电了。使用有线模式时，电量和连接状态也会通过灯条提示。

背光和侧灯

键盘背光：

  • Fn + ↑：提高背光亮度。
  • Fn + ↓：降低背光亮度。
  • Fn + ←：切换背光效果。
  • Fn + →：切换背光颜色。
  • Fn + <：降低背光速度。
  • Fn + >：提高背光速度。

侧边 RGB 灯：

  • Fn + M + ↑：提高侧灯亮度。
  • Fn + M + ↓：降低侧灯亮度。
  • Fn + M + ←：切换侧灯效果。
  • Fn + M + →：切换侧灯颜色。
  • Fn + M + <：降低侧灯速度。
  • Fn + M + >：提高侧灯速度。

如果想省电，可以把背光亮度调低，或者在无线模式下直接关掉灯效。

自动休眠

Air75 V2 支持自动休眠。开启后，键盘连续 6 分钟没有输入，会进入休眠状态，背光关闭。

开关休眠模式：

按下后看右侧 RGB：

  • 绿色闪 3 次：休眠模式已开启。
  • 红色闪 3 次：休眠模式已关闭。

无线使用时建议开启休眠，可以明显延长续航。如果你经常短时间离开电脑，回来后第一下按键没反应，多半是键盘刚从休眠唤醒。

截图快捷键

Air75 V2 有一个独立截图键。说明书里也列了全屏截图组合：

  • 截图键：区域截图。
  • Fn + 截图键：全屏截图。

如果你日常用 Snipaste、PixPin、CleanShot X 这类第三方截图工具，可以在 VIA 里把截图键改成对应软件的快捷键。例如 Snipaste 默认截图键是 F1，就可以把截图键映射为 F1。

查看电量

Air75 V2 可以临时显示电量状态：

触发后看右侧灯条颜色：

  • 红色：低电量。
  • 橙色：中等电量。
  • 绿色：接近满电或已满。

这个不是精确百分比，但足够判断是否该充电。

使用 VIA 改键

Air75 V2 支持 VIA，可以用网页配置器改键、保存配置和录制宏。

准备：

  1. 使用 Chrome、Edge 这类 Chromium 内核浏览器。
  2. 打开 VIA 在线配置器。
  3. 下载 NuPhy 官方 Air75 V2 JSON 文件：JSON files for NuPhy keyboards。
  4. 把键盘切到 Wired mode，用 USB-C 数据线连接电脑。

配置步骤：

  1. 在 VIA 顶部点齿轮图标，进入 SETTINGS。
  2. 打开 Show Design Tab。
  3. 点顶部画笔图标，进入 DESIGN。
  4. 上传 Air75 V2 的 JSON 文件。
  5. 回到 CONFIGURE，点击 Authorize Device。
  6. 在弹窗里选择 Air75 V2。

识别成功后，页面上会显示键盘布局。点选上方键盘里的某个按键，再在下方选择新的键值，就可以完成改键。VIA 的配置通常会直接写入键盘，不需要再手动保存。

VIA 的层对应关系

Air75 V2 的 Mac/Win 物理开关会对应到不同 VIA layer：

  • Mac mode：Layer 0 是默认层，Layer 1 是 Fn 层。
  • Win mode：Layer 2 是默认层，Layer 3 是 Fn 层。

如果你在 Mac 模式下改了键，切到 Windows 后发现没有生效，大概率是因为只改了 Layer 0/1，而 Windows 正在使用 Layer 2/3。

建议：

  • 只在 Mac 用：主要改 Layer 0/1。
  • 只在 Windows 用：主要改 Layer 2/3。
  • 两边都用：常用改键在两套 layer 里各改一遍。

录制宏

VIA 的宏要先录制，再绑定到按键。

  1. 在 CONFIGURE 页面左下选择 MACROS。
  2. 点击中间输入框上方的 </> 按钮，进入宏编辑模式。
  3. 选择一个宏槽，比如 M0。
  4. 输入宏内容。
  5. 点击 Save 保存。
  6. 回到 KEYMAPS，选择目标按键，在下方 MACRO 分类里选择 M0。

宏内容大致分两类：一类是直接输出的文本，一类是用 keycode 表示的按键或快捷键组合。NuPhy 官方说明里提到，可以在宏输入框输入 ? 搜索可用 keycode；不确定某个按键怎么写时，优先用 VIA 自带搜索，也可以查 QMK Keycodes 官方列表。

更新固件

固件更新不是日常必做项。只有遇到明显 bug，或者 NuPhy 发布了你确实需要的新功能时，再考虑更新。

大致流程：

  1. 下载 QMK Toolbox。
  2. 前往 NuPhy 固件发布页面下载 Air75 V2 对应固件，固件一般是 .bin 文件。
  3. 断开键盘和电脑的连接。
  4. 按住 Esc，同时插入 USB-C 数据线，让键盘进入刷写状态。
  5. 在 QMK Toolbox 里点击 Open 选择固件。
  6. 点击 Flash 开始刷写。

注意：刷固件过程中不要断开数据线。等 QMK Toolbox 显示完成后再拔线。

如果你使用 Apple Silicon Mac，通过 Homebrew 安装的 QMK Toolbox 遇到 Bad CPU type in executable 之类报错，可以尝试使用 QMK Toolbox 官方安装包版本。

小结

Air75 V2 日常使用最该记住的是三件事：

  1. 无线设备切换是 Fn + 1/2/3/4。
  2. 自动休眠开关是 Fn + ]，恢复出厂是长按 Fn + [。
  3. VIA 里 Mac 对应 Layer 0/1，Windows 对应 Layer 2/3。

把这几组操作记住后，连接、切换、改键和宏录制基本就顺了。
